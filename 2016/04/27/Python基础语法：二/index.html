Python基础语法二：条件选择与循环语句

发表于 2016-04-27   |   分类于 Python   |  

条件选择

  • if语句
i = 1
if i > 0 :
    print 'i大于0'
  • if else语句
i = 2
if i > 0 :
    print 'i大于0'
else :
    print 'i小于0'
  • if elif语句
i = 3             
if i == 4 :       
    print 'i等于4'  
elif i == 3 :     
    print 'i等于3'  
elif i == 2 :     
    print 'i等于2'

循环语句

  • for…in
code1：
for test in 'love' :      
    print '当前字母：' , test 

code2：
num = ['1','3','5','7']         
for index in range(len(num)):   
    print '当前数字：' , num[index]

输出结果：
当前字母： l
当前字母： o
当前字母： v
当前字母： e

  • while语句
i = 3                   
while (i < 30):         
    i = i+1             
else:                   
    print '跳出循环时的数字：',i

输出结果：30

  • while中的continue
i = 1            
while (i < 10):  
       i = i+1      
    if i%2 <> 0: 
        continue 
    else:        
        print i

输出结果：2 4 6 8

  • while中的break
i = 0                        
while (i < 5):               
    i += 1                   
    if i > 3:                
        print '大于3跳出，此时i=',i 
        break                
    else:                    
        print '打印小于等于3的数：',i

输出结果：
打印小于等于3的数： 1
打印小于等于3的数： 2
打印小于等于3的数： 3
大于3跳出，此时i= 4

  • pass语句
    Python pass是空语句，是为了保持程序结构的完整性。
    pass 不做任何事情，一般用做占位语句。
for i in range(1,10):                              
if i == 5 :                                    
    pass                                       
    print'卧槽pass语句块下面是5吧，没事继续往下打印'             
print i

输出结果：
1
2
3
4
