Python基础语法：一

发表于 2016-04-21   |   分类于 Python   |  

###一、print()函数###
print(‘Hello World’)

###二、raw_input()函数###
name=raw_input() #输入数据
print(‘这是手动输入的数据：’+name)

###三、format()函数###

# 5是占位,2是小数点后多少位，左右对齐取决于占位
print(format(12.345677,'5.2f'))

输出结果：12.34

###三、os.system()函数###
os.system(“pause”)
执行结果：在windows下让命令行窗口暂停

###四、类型转换###
str = ‘1’
print(int(str)+1)
输出结果：2

###五、id()函数###
说明：获取变量的内存id即地址

###六、变量###

  • 1、变量指向的是内存地址
  • 2、变量在定义时不需要指定数据类型
