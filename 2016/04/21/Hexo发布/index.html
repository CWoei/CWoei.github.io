Hexo个人博客如何进行文章发布

发表于 2016-04-21   |   分类于 Hexo   |  

打开Git控制台程序

D:\Program Files\Git\git-cmd.exe

进入Hexo工作目录

cd D:\StudyCode\CWoei.github.io

创建新文章

hexo new post postName

为文章加入分类和标签

---
title: about
date: 2016-04-21 17:13:06
categories:
- Hexo
tags:
- GitHub
- 博客
---

编辑好以后使用命令生成静态文件

hexo generate

部署到Git

hexo deploy
